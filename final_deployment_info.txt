## Known Issues and Solutions
- We found the `u256` parameters require a special format when using Starkli: `u256:{value}`
- For the MarketplaceSubscription contract deployment, we had to use this format for numeric parameters
- There is no explicit method in ComponentRegistry to set the DevSubscription address, highlighting a potential design oversight
